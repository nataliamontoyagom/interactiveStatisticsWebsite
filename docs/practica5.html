Práctica 5

Variables aleatorias y distribuciones continuas

1. Introducción

Con esta práctica se utiliza R y RStudio para calcular probabilidades de sucesos a partir de variables aleatorias continuas que tienen una distribución Normal o que se aproximan a una distribución Normal.

2. Variables continuas con distribución Normal

Una variable Normal es una variable continua (X) cuya función de probabilidad o densidad de probabilidad se ajusta a la siguiente expresión matemática, siendo μ la esperanza (media) y σ la desviación estándar de la variable:

\[f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^{2}}{2\sigma^{2}}}\]

Se representa como X:N(μ,σ) y se dice que 𝑋 tiene una distribución Normal.

Por ejemplo, una variable X:N(5,1) que representa: “Calificación obtenida en una asignatura con una calificación media de 5 y una desviación estándar de 1”.

En R se pueden utilizar las funciones indicadas en la siguiente tabla. Por simplicidad, para la esperanza “μ” usaremos “m” y para la desviación estándar usaremos “d”. En la columna “Ejemplo de uso”, se utiliza una variable N(5,1).

2.3 Aproximar una variable Binomial o de Poisson a una distribución Normal

1) Aproximar una variable Binomial a una distribución Normal

Si en una variable Binomial X:B(n,p) se cumple que n es grande4, entonces se puede aproximar a una variable Normal X:N(m,d), con \(m = np\) y \(d = \sqrt{np(1-p)}\), y se puede calcular que la variable Binomial esté en el intervalo [a,b] calculando P{X≤b}-P{X<a} aplicando la Normal, pero en este caso sería P{X<b+0.5}-P{X<a-0.5}, porque hay que expandir el intervalo sumando y restando 0.5 en cada extremo debido a la conocida corrección de continuidad, debido a que la distribución Binomial es discreta y la Normal continua.

Se puede comprobar por ejemplo con B(100,0.5), aproximándola a una Normal N(50, 5).

3.1 Ejemplo: Correo electrónico no deseado (spam)

ENUNCIADO

Una persona ha comprobado que recibe una media de 48 mensajes al día de correo electrónico no deseado (spam).

  1. Definir una variable aleatoria Exponencial para el tiempo entre mensajes.
  2. Calcular la esperanza de la variable aleatoria.
  3. Cuál es la probabilidad de que el tiempo entre dos mensajes sea de menos de 1 hora.
  4. Cuál es la probabilidad de que el tiempo entre mensajes sea de más de 1 hora.
  5. Cuál es la probabilidad de que el tiempo entre mensajes esté entre 1 y 2 horas.

SOLUCIÓN

  1. Definir una variable aleatoria Exponencial para el tiempo entre mensajes.

X = Tiempo entre dos mensajes no deseados recibidos, sabiendo que la media de mensajes no deseados recibidos en una hora es de 48/24=2 mensajes.

NOTA: Como las preguntas se refieren a horas se ha hecho la conversión de días a horas.

La variable sigue una distribución Exponencial(λ), donde λ=48/24=2 mensajes/hora. Por simplicidad se utiliza “lambda” en lugar de “λ” en el código R.

  1. Calcular la esperanza de la variable aleatoria.

La esperanza es 1/λ horas entre dos mensajes consecutivos.

  1. Cuál es la probabilidad de que el tiempo entre dos mensajes sea de menos de 1 hora.

Se trata de calcular P{X<1} que al ser una variable continua equivale a P{X≤1}. Se puede usar, por tanto, la función de distribución acumulada pexp():

  1. Cuál es la probabilidad de que el tiempo entre mensajes sea de más de 1 hora.

En ese caso P{X>1} = 1 – P{X<1}.

4.1 Variables continuas con distribución general

Si dada una variable aleatoria continua de la que disponemos de un conjunto de valores, no encontramos ninguna distribución conocida (Normal, Beta, Gamma, etc.) a la que se ajuste, entonces se pueden utilizar las funciones density() y ecdf() para calcular empíricamente probabilidades relacionadas con variable.

Toda variable aleatoria continua X tiene asociada dos funciones:

  • La función de densidad de probabilidad f(x): que representa la probabilidad de que la variable tenga un valor concreto: P{X=x}. En el caso de las variables continuas, esta función sólo se utiliza a efectos teóricos y de representación gráfica, porque la probabilidad de la variable en un punto no puede calcularse, es cero, y siempre hay que calcular la probabilidad en un intervalo. En el caso de R se puede asociar a la función density().

  • La función de distribución acumulada F(x), que representa la probabilidad de que la variable sea igual a inferior a un valor concreto: P{X≤x}. En una variable continua, al no tener sentido hablar de probabilidad de un valor, en este caso se cumple que P{X≤x}=P{X<x}. En el caso de R se puede asociar a la función ecdf().

Si disponemos de una muestra de valores de la variable, con R se pueden manejar ambas funciones como se indica en la siguiente tabla.
