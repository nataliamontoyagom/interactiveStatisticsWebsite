# README
Este proyecto trata del desarrollo de un sitio web interactivo para el aprendizaje de análisis estadístico con R.
Se utiliza Quarto como herramienta fundamental sobre la que se basa el sitio web, el cual contendrá una serie de prácticas disponibles actualmente en: https://hilera.web.uah.es/estadistica/r/index.html
El proyecto estará público en GitHub Pages.
